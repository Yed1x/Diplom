# Руководство администратора системы классификации шахматных фигур

## 1. Общие сведения
### 1.1 Назначение и область применения
Система предназначена для автоматической классификации шахматных фигур с использованием технологий глубокого обучения. Система может быть развернута как веб-приложение (Streamlit) или как десктопное приложение (PyQt).

### 1.2 Условия эксплуатации
- Операционная система: Windows 10/11, Linux, macOS
- Python 3.8 или выше
- Минимум 4 ГБ оперативной памяти
- Поддержка CUDA для ускорения работы нейронной сети (опционально)

## 2. Структура системы
### 2.1 Компоненты системы
- `app.py` - веб-интерфейс на Streamlit
- `chess_classifier_gui.py` - десктопное приложение на PyQt
- `final_model.h5` - обученная модель нейронной сети
- `predictions_log.csv` - файл логов предсказаний

### 2.2 Требования к программному обеспечению
```
tensorflow>=2.8.0
streamlit>=1.10.0
pillow>=8.0.0
numpy>=1.19.0
pandas>=1.3.0
matplotlib>=3.4.0
PyQt5>=5.15.0
```

## 3. Установка и настройка
### 3.1 Установка зависимостей
```bash
pip install -r requirements.txt
```

### 3.2 Настройка веб-приложения
1. Убедитесь, что файл `final_model.h5` находится в корневой директории
2. Запустите приложение командой:
```bash
python -m streamlit run app.py
```

### 3.3 Настройка десктопного приложения
1. Убедитесь, что файл `final_model.h5` находится в корневой директории
2. Запустите приложение командой:
```bash
python chess_classifier_gui.py
```

## 4. Администрирование
### 4.1 Управление логами
- Файл `predictions_log.csv` содержит историю всех предсказаний
- Рекомендуется периодически архивировать старые логи
- Максимальный размер файла логов: 100 МБ

### 4.2 Обновление модели
1. Создайте резервную копию текущей модели
2. Замените файл `final_model.h5` новой версией
3. Перезапустите приложение

### 4.3 Мониторинг
- Проверяйте размер файла логов
- Следите за использованием памяти
- Контролируйте точность предсказаний

## 5. Безопасность
### 5.1 Доступ к системе
- Веб-приложение доступно только локально
- Десктопное приложение не требует специальных прав доступа

### 5.2 Защита данных
- Регулярное резервное копирование модели
- Архивация логов
- Защита от несанкционированного доступа к файлам

## 6. Устранение неполадок
### 6.1 Типичные проблемы
1. Ошибка загрузки модели
   - Проверьте наличие файла `final_model.h5`
   - Убедитесь в совместимости версий TensorFlow

2. Проблемы с логами
   - Проверьте права доступа к файлу
   - Убедитесь в корректности кодировки

3. Проблемы с памятью
   - Увеличьте размер файла подкачки
   - Закройте неиспользуемые приложения
